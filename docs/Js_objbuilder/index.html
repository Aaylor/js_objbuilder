Module Js_objbuilder

This library allows to write javascript objects with optional values easier.

type 'a t

The type representation of an object builder. The 'a type correspond to the result js object type.

val empty : _ t

The empty builder.

Basic values

val push : string ‑> ('a ‑> Js.Unsafe.any) ‑> 'a option ‑> 'b t ‑> 'b t

push key translater ovalue acc checks if the ovalue is set, and in this case use translater to store the value, associated with key to the builder acc.

val push_bool : string ‑> bool option ‑> 'a t ‑> 'a t

push_bool key obool acc pushes the boolean value to acc if it is set.

val push_float : string ‑> float option ‑> 'a t ‑> 'a t

push_float key ofloat acc pushes the float value to acc if it is set.

val push_int : string ‑> int option ‑> 'a t ‑> 'a t

push_int key oint acc pushes the int value to acc if it is set.

val push_string : string ‑> string option ‑> 'a t ‑> 'a t

push_string key ostring acc pushes the string value to acc if it is set.

Optional values

val push_option : string ‑> ('a ‑> Js.Unsafe.any) ‑> 'a option option ‑> 'b t ‑> 'b t

push_option key translater ovalue acc is similar to push, excepts that Some None is translated into the javascript null value.
